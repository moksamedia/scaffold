# Conventions and workflow

- All Vue code uses the **Composition API** with `<script setup>`.
- Keep **README.md** updated with install/run steps and noteworthy features.
- Keep **CLAUDE.md** (this tree) aligned with actual structure—update the relevant shard when you change architecture.

## Git discipline

- Before starting new feature work, ensure a **clean working tree** (`git status`).
- Commit **before and after** large changes or new features where practical.
- After completing a coherent change set, **commit before** moving to unrelated work.
